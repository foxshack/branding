Welcome to Semantic Styling

This page demonstrates a comprehensive semantic CSS theme built with your brand colors: Solar Clay, Mineral Grey, and Verdant Lime. Every HTML element is styled based on its semantic meaning, not arbitrary classes.

What's Semantic CSS?

Semantic CSS means styling HTML elements directly so that your markup structure conveys meaning without needing class names. This approach keeps HTML cleaner and CSS more maintainable.

Code & Preformatted Text

Inline Code

For short code snippets in text, use code elements like this: const theme = { primary: '#D96E3C' }.

Code Blocks

For longer code examples, use <pre> elements:

/* Define semantic color tokens */
:root {
  --fs-color-primary: #D96E3C;
  --fs-color-text: #333333;
  --fs-color-background: #FFFFFF;
}
